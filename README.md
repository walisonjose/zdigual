# Teste Desenvolvedor Pleno Zdigual
Repositório criado para realização do teste de Desenenvolvedor Pleno na empresa Zdigual.

# Implantando a aplicação
###### Clonando o repositório
* git clone https://github.com/walisonjose/zdigual.git <dir_aplicacao>
* cd <dir_aplicacao>
 ###### Instalando as dependencias
* npm install
 ###### Realizando o build
* npm run build
###### Executando a aplicação
* npm start
###### Acessando a aplicação
* Abra o navegador e acesse o endereço [http://localhost:3001/](http://localhost:3001/)
